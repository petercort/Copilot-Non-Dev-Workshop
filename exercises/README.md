# Hands-on Labs (Copilot Chat)

These labs are designed for **GitHub Copilot Chat** (VS Code) and are each **30–60 minutes**.

## Labs

1. [`exercises/01-business-analyst-lab.md`](01-business-analyst-lab.md) — Discovery → user stories → doc comparison
2. [`exercises/02-product-manager-lab.md`](02-product-manager-lab.md) — Backlog → roadmap → sprint status → release comms
3. [`exercises/03-systems-analyst-lab.md`](03-systems-analyst-lab.md) — Explain code/data → ADR → sequence diagram
4. [`exercises/04-qa-test-lab.md`](04-qa-test-lab.md) — AC → test cases → bug triage → risk-based testing

## How to run a lab

1. Open this repo in VS Code.
2. Open **GitHub Copilot Chat**.
3. Follow the lab steps. Each step points you to a prompt in `prompts/` and an input file in `scenario/`.

Tip: If your Copilot Chat supports referencing workspace files, you can attach or reference the files in `scenario/` instead of pasting them.
